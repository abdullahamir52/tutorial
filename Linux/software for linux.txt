=========================================================================
			    	 Avro Keyboard
=========================================================================
1. Open Ubuntu Software
2. Search "Avro Phonetic" in Ubuntu Software
3. Click "Install"
4. Open "Settings" > "Region & Language"
5. Click on the "+" (plus) icon in Input Sources
6. Search for "Bangla"
7. click on "Other" and select "Bangla (Avro Phonetic)"
   The Avro Phonetic will be added to the Input Sources
8. You can switch between different Input Source using "win + space" 


=========================================================================
		    Pinta (Microsoft Paint Alternative)
=========================================================================
Install Pinta from Pinta maintainers PPA repository using terminal 
(for advanced users)

1. Open terminal 
CTRL+ALT+T

2. Add Pinta stable PPA repository:
sudo add-apt-repository ppa:pinta-maintainers/pinta-stable
Note: Instead of pinta-stable you can add pinta-daily if you like to 
test latest and the greatest Pinta, but it may be buggy.

3. Update system package lists:
sudo apt-get update

4. Install pinta:
sudo apt-get install pinta

# ----------------
# Installing R 

# Install the dependencies necessary to add a new repository over HTTPS:
sudo apt install dirmngr gnupg apt-transport-https ca-certificates software-properties-common

# Add the CRAN repository to your system sourcesâ€™ list:
sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-keys E298A3A825C0D65DFD57CBB651716619E084DAB9
sudo add-apt-repository 'deb https://cloud.r-project.org/bin/linux/ubuntu focal-cran40/'

# Install R by typing:
sudo apt install r-base

# The installation may take a few minutes to complete. Once completed, verify it by printing the R version:
R --version
# ----------------


# ----------------
# Uninstalling R
# Uninstall just r-base

sudo apt-get remove r-base

# This will remove just the r-base package itself.
# ----------------

# ----------------
# Uninstall r-base and its dependencies

sudo apt-get remove --auto-remove r-base

# This will remove the r-base package and any other dependant packages which are no longer needed.
# ----------------

# ----------------
# Purging your config/data too
# If you also want to delete your local/config files for r-base then this will work.

# Caution! Purged config/data can not be restored by reinstalling the package.

sudo apt-get purge r-base

# Or similarly, like this r-base
sudo apt-get purge --auto-remove r-base
# ----------------
sudo apt-get remove r-base-core
sudo apt-get autoremove
